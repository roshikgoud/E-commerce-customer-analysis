# E-commerce-customer-analysis
🛒 E-Commerce Customer Analysis
An end-to-end Exploratory Data Analysis (EDA) on a Turkish e-commerce platform dataset, uncovering actionable insights across revenue, customer retention, device behavior, payment methods, and geographic concentration — culminating in an executive-ready dashboard.

## 📊 Project Overview
This project dives deep into customer behavior and business performance on a Turkish e-commerce platform. The goal was to surface patterns that business stakeholders can act on, presented through clean visualizations and a condensed executive dashboard.

## 🔍 Key Findings
| Area | Insight |
|------|---------|
| Revenue | Identified high-revenue segments and seasonal trends |
| Customer Retention | Analysed repeat purchase behavior and churn signals |
| Device Behavior | Compared conversion rates across mobile vs desktop users |
| Payment Methods | Mapped payment preferences to average order value |
| Geography | Pinpointed regional concentration of high-value customers |
